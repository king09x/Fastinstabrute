# FastInstagramBrute

Fake Instagram login page with Python-based brute-force tester using wordlist.

## Setup

### 1. Run PHP Server

```bash
cd backend
php -S 127.0.0.1:8000
```

### 2. Open frontend/login.html in browser

### 3. Run Brute Force

```bash
python3 brute.py
```
